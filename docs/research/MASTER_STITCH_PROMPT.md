# ResQLink: The "Feature-Deep" Master Stitch Prompt

This prompt is engineered for **Google Stitch** to generate a high-fidelity prototype landing page. It prioritizes **explicit feature explanation** over simple linking, ensuring the user understands the "How" and "Why" of the ResQLink SaaS platform within a single-page view.
